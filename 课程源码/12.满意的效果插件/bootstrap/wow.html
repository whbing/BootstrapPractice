WOW.js 使用教程

简单实用，可能还要加一个炫酷，才能表达出WOW的本性！

1、引入animate.css文件

<link rel="stylesheet" href="css/animate.min.css">

2、引入wow.js文件

<script src="wow.min.js"></script>

3、JavaScript

new WOW().init();

4、自定义配置(可选)

var wow = new WOW({
boxClass: 'wow',
animateClass: 'animated',
offset: 0,
mobile: true,
live: true
 });
wow.init();
